📡 Data Sources & Methodology

Each company is fetched individually from their own public status API directly in your browser — there is no server or aggregator. 28 HTTP requests are made in parallel every 60 seconds (2 per company: one for current status, one for incident history).

Uptime % is calculated from the last 50 incidents returned by each API over the past 90 days. For each incident (excluding impact: none), the actual resolved duration is subtracted from the 90-day window. Ongoing incidents count as down until now. The API caps at 50 incidents per request — companies with very high incident frequency may show slightly inflated uptime figures.

Most companies in this list use Statuspage.io (owned by Atlassian), which provides a standard REST API at a consistent URL pattern — this is why all endpoints look identical.
